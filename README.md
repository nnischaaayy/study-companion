## 📸 What It Does

Study Companion runs silently in the background while you work. It:

- **Watches your active window** every 10 seconds and classifies it as Focused, Distracted, Neutral, or Break
- **Takes a screenshot** every 60 seconds and uses AI vision to deeply analyze what you're actually looking at (catches whether a YouTube video is educational or entertainment, reads browser content, identifies what you're coding)
- **Tracks typing activity** to distinguish active work from idle screen-staring
- **Links every activity to a specific task** from your daily task board
- **Auto-detects completed tasks** at the end of a session based on your activity
- **Generates a full AI report** at session end with time breakdown, distraction log, productivity score, and recommendations
- **Shows a floating Pomodoro timer** that stays on top of every window

| Packaging  | PyInstaller + Inno Setup | Compiles to a single .exe with a proper Windows installer   |
| Updates    | GitHub Releases API      | Auto-update banner when new version is available            |

## ✨ Features

### 🎯 Session Tracking

- Set a session goal and detailed context before starting
- Live activity feed showing every window switch and screenshot analysis
- Real-time focus score (0–100%) updating as you work
- Elapsed session timer
- Color-coded status: 🟢 Focused / 🔴 Distracted / 🟡 Neutral / ⚪ Break

### ✅ Task Board

- Add tasks for the day with category (Study / Code / Research / Admin), estimated time, and notes
- AI automatically links your activity to specific tasks throughout the session
- Tasks auto-marked as completed or in-progress at session end based on your activity
- Carry-over feature — incomplete tasks appear again the next day
- Manual check-off, edit, and delete

### 📋 Session Reports

- Generated by Gemini at session end
- Includes: Session Summary, Task Completion Report, Time Breakdown, Focus Analysis, Distraction Log, Productivity Score, Recommendations
- Saveable as a Markdown file

### ⏱ Pomodoro Overlay

- Slim draggable bar that floats above every window
- 25min work → 5min short break → 15min long break cycles
- Shows live focus status and score from the current session
- Run separately alongside the main app

### 🔄 Auto-Updates

- Checks GitHub Releases on every launch
- Shows a non-intrusive banner when a new version is available
- One click opens the download page

## 🚀 Installation (For Users)

1. Go to the [Releases page](https://github.com/nnischaaayy/study-companion/releases)
2. Download `StudyCompanion_Setup_vX.X.X.exe`
3. Run the installer — it adds the app to your Start Menu and taskbar
4. Launch **Study Companion** from Start Menu
5. Get a free Gemini API key from [aistudio.google.com/apikey](https://aistudio.google.com/apikey) (no credit card needed)
6. Paste the key in the sidebar — it's saved automatically after that

**No Python, no pip, no terminal required.**

## 🔑 Getting a Free Gemini API Key

1. Go to [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Sign in with your Google account
3. Click **Create API key**
4. Copy the key and paste it into the app sidebar

**Free tier limits:** 1,500 requests/day, 1M tokens/day — more than enough for full day usage. A typical 8-hour session uses ~550 requests.

## 💻 Running From Source (For Developers)

### Prerequisites

- Python 3.10 or higher
- Windows 10 / 11
- A free Gemini API key

### Setup

```bash
# Clone the repo
git clone https://github.com/nnischaaayy/study-companion.git
cd study-companion

# Install dependencies
pip install -r requirements.txt

# Fix pywin32 if needed
python -m pywin32_postinstall -install

# Run the app
python main.py

# Run the Pomodoro overlay (separate terminal)
python overlay.py
```

## 🏗 Building the App

### Prerequisites

- [PyInstaller](https://pyinstaller.org/) — installed automatically by build.bat
- [Inno Setup 6](https://jrsoftware.org/isinfo.php) — install manually once
- [GitHub CLI](https://cli.github.com/) — for publishing releases

### One-time setup

```bash
# Authenticate GitHub CLI
gh auth login
```

### Build without releasing

```bash
.\build.bat
```

Output: `dist\StudyCompanion.exe` and `installer_output\StudyCompanion_Setup_v*.exe`

### Build and release

```bash
.\release.bat 1.0.5
```

This single command:

1. Updates `version.py` to the new version
2. Generates the app icon
3. Builds the `.exe` with PyInstaller
4. Builds the installer with Inno Setup
5. Commits and tags in git
6. Pushes to GitHub
7. Publishes a GitHub Release with the installer attached

Users already running the app will see an update banner automatically.

## 🗄 Database Schema

All data is stored locally in `study_sessions.db` (SQLite).

```sql
sessions   — id, task, context, start_time, end_time, report
events     — session_id, ts, app_name, window_title, classification,
             confidence, reason, task_id, task_name, typing_active, event_type
tasks      — id, title, category, estimated_mins, status, created_date,
             completed_date, session_id, carry_over, notes, time_spent_s
```

## 🔒 Privacy

- **Screenshots** are sent to Google's Gemini API for classification only — not stored by this app
- **Keyboard monitoring** detects whether typing is happening — no keystrokes are recorded or transmitted
- **Window titles** are stored locally in `study_sessions.db` only
- **API key** is stored locally in `.gemini_key` — never sent anywhere except Google's API
- **No telemetry** — the app never phones home or tracks usage
